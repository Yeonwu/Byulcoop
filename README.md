# Byulcoop
별무리학교 매점 재고 관리를 위한 시스템

언젠가 이 코드를 보수하게 될 누군가에게

만약 당신이 이 정도 규모의 프로젝트를 진행해 본적이 있다면 그냥 다시 짜는게 빠를겁니다.

만약 당신이 이 정도는 아니더라도 프로젝트를 진행해 본적이 있다면 이 코드를 읽어만 보고 새로 짜는게 빠를겁니다.

만약 당신이 처음 프로젝트를 시작한다면... 그냥 하지 않는게 나을 수도 있습니다.


우리도 처음으로 진행한 프로젝트이기 때문에 코드가 굉장히 더럽습니다. 일관성도 없고 다 제각각입니다.
